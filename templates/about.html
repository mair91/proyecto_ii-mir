Mexico's Presidential Elections Overview 1994-2018


This site aims to provide an overview of the last 5 mexican presidential elections, displaying a dashboard with the general results of the elections using the databases provided by the Federal Electoral Institute.

This site is the result of the second project for the Tec de Monterrey Data Bootcamp and uses HTML, JavaScript, CSS, Leaflet, SQL.
